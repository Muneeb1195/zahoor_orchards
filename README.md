# Zahoor Orchards

A premium housing society website — built with Next.js, deployed to GitHub Pages.

## Tech Stack

- **Next.js 16** (App Router, static export)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** (scroll-linked animations)
- **lucide-react** (icons)

## Sections

1. **Hero** — full-width banner with scroll-driven logo shrink
2. **About** — society details with stats
3. **Amenities** — park, security, roads, sewerage, mosque, hospital
4. **Gallery** — image grid with fullscreen modal + navigation
5. **Plots** — available/sold plots table
6. **Payment Plan** — pricing tiers with download
7. **Management** — team members
8. **News** — latest updates
9. **Contact** — inquiry form, phone, social links

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Static output in `out/`.

## Deployment

Push to `main` — GitHub Actions builds and deploys to:

**https://muneeb1195.github.io/zahoor_orchards/**
